Retention Policy
1. Ограничение по размеру

default: -1
2. Ограничение по времени
retention.ms
default: 7 дней

Cleanup Policy
Delete
   Удалять сегменты при истечении срака их хранения, либо при достижении предельного размера
Compact
   Включает log compaction, который сохраняет последнее значение каждого ключа

Политики не взаимоисключающие.
